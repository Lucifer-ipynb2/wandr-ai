# ✈️ Wandr — AI Travel Planner

A beautiful, production-ready AI travel planning app built with **React + Vite**, powered by **GPT-4o via OpenRouter**.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 🔑 Setup

1. Get a free API key at [openrouter.ai/keys](https://openrouter.ai/keys)
2. Paste your `sk-or-v1-...` key in the app modal
3. Start planning your dream trip!

## ✨ Features

- **AI Itinerary Generator** — Full day-by-day travel plans powered by GPT-4o
- **Smart Form** — Pick destination, duration, budget, interests & travel style
- **Rich Itinerary View** — Morning / afternoon / evening breakdown, meals, tips
- **AI Chat Assistant** — Ask any travel question in real-time
- **PDF Save** — Print/save your itinerary directly from the browser
- **Secure** — API key stored locally, never sent to any server

## 🛠 Tech Stack

- React 18 + Vite 5
- CSS Modules (no CSS-in-JS)
- OpenRouter API (GPT-4o)
- Lucide React icons
- Google Fonts (Cormorant Garamond + DM Sans)

## 📦 Build

```bash
npm run build
```

Output in `dist/` — ready to deploy to Vercel, Netlify, or any static host.

## 🌐 Deploy to Vercel

```bash
npx vercel --prod
```
